## 🌱백엔드
백엔드에서는 spring boot, jpa, mariaDB를 사용하였다. 로그인과 회원가입의 보안을 위해 security를 이용하였고,
MVC model을 사용하여 서버와 프론트를 thymeleaf 통해 연결했다.
